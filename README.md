🏠 서울 부동산 임대차 데이터 분석 대시보드 (Streamlit)📌 

1. 프로젝트 개요이 프로젝트는 서울시 전월세 계약 데이터(seoul.csv)를 기반으로 구축된 대화형 Streamlit 대시보드입니다. 
사용자에게 핵심 시장 지표, 가격 효율성 분석, 시장 트렌드, 그리고 잠재적 리스크에 대한 심층적인 분석 결과를 제공하여 데이터 기반의 합리적인 부동산 의사결정을 지원하는 것을 목표로 합니다.

✨ 2. 주요 기능 요약대시보드는 네 가지 핵심 분석 페이지로 구성되어 있습니다.페이지핵심 목표주요 분석 내용Home시장 현황 및 KPI 요약총 계약 건수, 평균 보증금/임대료, 최고가 및 최저가 거래 즉시 확인.
1. 상세 분석 대시보드가격 효율성 및 분포 분석**1㎡당 가격 (평당 가격)**을 계산하여 자치구별 효율성 비교 및 비효율적 거래 탐색.
2. 시장 트렌드 분석시간 경과에 따른 변화 추이선택된 지역/연도의 월별 평균 가격 및 계약 건수 변화 추이 분석.
3. 심화 맞춤 비교 분석가상 시장 그룹 비교 (A vs B)사용자가 정의한 두 그룹(예: '강남구 아파트 전세' vs '강북구 연립 월세')의 가격, 면적, 회귀선 기울기 비교.
4. 리스크 및 노후도 분석잠재적 위험 요소 포착통계적 이상치(Outlier) 탐색으로 가격 리스크 경고 및 건물 노후도에 따른 리스크 평가.

⚙️ 3. 실행 방법 (How to Run)이 대시보드를 로컬 환경에서 실행하려면 Python과 Streamlit 환경이 필요합니다.
3-1. 환경 설정필요 라이브러리 설치:Bashpip install streamlit pandas plotly numpy
데이터 파일 준비:분석에 사용할 seoul.csv 파일을 프로젝트의 루트 디렉토리에 위치시킵니다. (파일명 불일치 시 app.py의 FILE_PATH 변수를 수정해야 합니다.)파일 구조 확인:프로젝트 디렉토리는 최소한 다음과 같은 구조를 가져야 합니다.real_estate_dashboard/
├── app.py             # 홈 화면 및 데이터 로드 정의
├── seoul.csv          # 원본 데이터 파일
└── pages/
    ├── 1_Analysis_Dashboard.py
    ├── 2_Market_Trend_Analysis.py
    ├── 3_Comparative_Analysis.py
    └── 4_Risk_and_Forecast.py
3-2. 대시보드 실행터미널에서 프로젝트 루트 디렉토리로 이동한 후, 다음 명령어를 실행합니다.Bashstreamlit run app.py
명령어 실행 후 웹 브라우저가 열리며 대시보드가 나타납니다.📝 4. 사용된 기술 스택언어: Python프레임워크: Streamlit데이터 처리: Pandas시각화: Plotly (Express, Graph Objects)
